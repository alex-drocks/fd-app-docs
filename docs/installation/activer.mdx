Pour activer le <NomLogiciel/>, copiez-collez votre clé d'activation
dans l'interface du logiciel prévue à cet effet. Puis appuyez sur Enter.

## 1. Copiez votre clé d'activation
Avec votre souris, sélectionnez le texte de la licence au complet.

Ensuite, faites un clic droit avec la souris et choisissez "Copier".
Ou, au clavier, appuyez sur <kbd>Control</kbd> + <kbd>C</kbd>.

<Image img={cleActivation}/>

Note :
Votre licence a été reçue lors de la confirmation de votre
[commande du logiciel Finance D](https://magasin.finance-d.com/mon-compte/orders/).
Elle se retrouve aussi dans votre courriel de confirmation.

## 2. Collez-la dans le logiciel
Une fois que vous avez copié la clé d'activation,
retournez dans le logiciel Finance D.
Assurez-vous que votre curseur pour écrire est dans la case de la clé d'activation.
Ensuite, avec votre clavier, appuyez sur <kbd>Control</kbd> + <kbd>V</kbd> pour coller la licence dedans.

<Image img={screenshot} />

<br/>

## Terminé
Si votre clé d'activation est validée, une confirmation s'affichera dans le logiciel vous indiquant
de procéder à la prochaine étape qui est de
[créer votre premier dossier de tenue de livres sécurisé](/docs/bases/nouveau-dossier)
(cela prend 30 secondes).
