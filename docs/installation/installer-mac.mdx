id: installer-mac
title: Installation du logiciel sur ordinateurs Mac
sidebar_label: Installer sur Mac
---


Après avoir téléchargé le logiciel, double-cliquez sur le fichier d'installation et attendez
de voir la fenêtre où votre Mac permettra de glisser le logiciel dans vos Applications.
Si vous avez exécuté cette action correctement,
vous devriez voir un message similaire à
"Copie de Finance D dans vos Applications".
<br/>
Lorsque l'installation est terminée, ouvrez l'application comme vous faites habituellement.
Le logiciel va d'abord s'ouvrir à la fenêtre d'activation où vous pourrez saisir votre clé de licence.
