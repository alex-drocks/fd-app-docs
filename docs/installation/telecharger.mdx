## Vidéo YouTube
Si vous préférez visionner un court vidéo d'explication étape par étape, voici le lien YouTube

https://youtu.be/9364t0Y2XmY

## Cliquez sur le bouton télécharger dans la page de confirmation
Lorsque vous avez reçu la confirmation de votre
[commande du logiciel Finance D (gratuit)](https://magasin.finance-d.com/produit/logiciel-tenue-de-livres-simplifiee),
cliquez sur le bouton TÉLÉCHARGER qui correspond à votre ordinateur. Soit Windows, ou Mac.

<Image img={confirmationCommande}/>

<br/>
<br/>

<hr/>

## Alternative : Cliquez sur le bouton télécharger du courriel de confirmation

Vous êtes également supposé avoir reçu la confirmation de commande par courriel.
Donc, vous pourriez télécharger le logiciel Finance D à partir de là
si vous préférez (le résultat sera le même).

<Image img={confirmationCourriel}/>

<br/>
<br/>
<hr/>

Rendu à cette étape, vous devriez voir le fichier d'installation du
[logiciel Finance D](https://finance-d.com) à
l'endroit habituel où votre navigateur internet les place.
Avec ce fichier d'installation, vous seriez donc prêts à installer le logiciel sur votre ordinateur.
