Pour créer un nouveau dossier, choisissez un nom (par exemple, le nom de votre entreprise),
un mot de passe pour protéger vos données, et votre adresse courriel pour récupérer le mot de passe si
jamais vous l'oubliez.

<Image img={screenshot}/>
<br/>
Ensuite, cliquez sur le bouton "Créer le dossier".
