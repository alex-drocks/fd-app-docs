Pour créer une facture de revenus, il suffit d'entrer le nom de votre client, la date de la facture
et le montant que vous lui chargez (avec ou sans taxes dépendamment de votre situation).

<Image img={screenshot}/>
<br/>
Lorsque la facture est complète, cliquez sur le bouton "ENREGISTRER LA FACTURE"
pour la conserver et pouvoir en créer une nouvelle.

<br/>
<br/>
<hr/>
Si vous voulez imprimer la facture, cliquez sur le logo d'imprimante
pour allez à la fenêtre permettant d'imprimer vos factures de revenus.
Notez qu'en cliquant sur le bouton d'impression, la facture est automatiquement
sauvegardée donc pas besoin d'appuyer sur le bouton "ENREGISTRER LA FACTURE" avant.
