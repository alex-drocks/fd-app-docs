Pour imprimer une facture de revenus, il faut d'abord être dans une facture de revenus
valide.
<br/>
Ensuite, cliquez sur le bouton d'imprimante
pour aller à la fenêtre permettant d'imprimer vos factures de revenus.
<br/>
<br/>

:::note
Notez qu'en cliquant sur le bouton d'impression, la facture est automatiquement
sauvegardée donc pas besoin d'appuyer sur le bouton "ENREGISTRER LA FACTURE" avant.
:::

<br/>
<Image img={screenshot}/>

<br/>
<hr/>
<br/>

Lorsque la fenêtre d'impression s'affiche pour la première fois,
ce sera sans doute un peu intimidant car il faut configurer vos informations personnelles.

:::note
En passant votre souris sur les éléments, une icône apparaitra pour vous permettre de modifier
vos informations en cliquant.
:::

<Image img={screenshot2}/>
<br/>
<br/>
Cette étape plus longue n'est à faire qu'une seule fois. Toutes vos prochaines
factures incluront les détails que vous avez ajoutés (vous pourrez aussi les modifier au besoin).
<br/>
<br/>
<Image img={screenshot3}/>
<br/>
<br/>

<Image img={screenshot4}/>

<br/>
<hr/>
<br/>

Lorsque vous aurez saisi vos informations pour afficher sur votre facture,
votre modèle devrait ressembler à cela.
Vous pourrez maintenant imprimer la facture en utilisant l'un des boutons bleus à droite
de la fenêtre. Il est possible d'imprimer en PDF, en papier ou d'envoyer un PDF par courriel.
<Image img={screenshot5}/>
<br/>

:::info
Pour l'envoi par courriel, vous devez avoir fourni l'adresse courriel du client.
Dans le cas contraire, le bouton ne sera pas visible.
:::
